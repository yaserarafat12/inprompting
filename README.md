# ✦ Inprompting — Belajar Ngomong ke AI

**Inprompting** adalah alat bantu (tool) berbasis web untuk membangun prompt AI yang efektif menggunakan framework **RTCFOS** (Role, Task, Context, Focus, Output, Stop).

## 🚀 Fitur Utama
- **Interactive Builder**: Membangun prompt secara dinamis dengan pilihan Role, Task, dan Format.
- **AI-Powered Suggestions**: Mendapatkan saran otomatis dari Anthropic Claude untuk memperkuat prompt Anda.
- **Modern UI**: Interface chat yang responsif dengan tema estetik, noise texture, dan micro-animations.
- **Templates**: Pilihan template cepat untuk YouTube, TikTok, Coding, dan Analisis Dokumen.

## 🛠 Tech Stack
- **Frontend**: React.js + Vite
- **Logic**: RTCFOS Framework Integration
- **API**: Anthropic API (Claude 3.5 Sonnet)
- **Styling**: Vanilla CSS (Custom UI System)

## 📦 Cara Menjalankan Lokal

1. Clone repository ini.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Jalankan server development:
   ```bash
   npm run dev
   ```
4. Buka di browser: `http://localhost:5173`
